# SWC Documentation

SWC lets you build reactive web components using nothing but modern browser APIs. State changes re-render your components automatically. Styles are scoped. No build step required.

## How to read these docs

If you're new, start at the top and read in order. Each section builds on the previous one.

If you already know web components and just want to get something running, jump straight to [Getting Started](getting-started/README.md).

If you're looking for a specific method or option, check the [API Reference](api/README.md).
