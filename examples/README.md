# SWC Examples

Fully built examples that demonstrate different ways of using the SWC library. Each example
is self-contained — open `index.html` (or `index.php`) in a local server and it runs.

| Example | What it shows |
| :--- | :--- |
| [`shared-state/`](shared-state/) | Two components, one store — reactive sync across the page |
| [`php-ssr/`](php-ssr/) | Server-side rendering with the PHP package |
| [`router/`](router/) | Client-side SPA routing with active links and dynamic params |
| [`theme-switcher/`](theme-switcher/) | Global theme state propagating to multiple components |
| [`custom-renderer/`](custom-renderer/) | Plugging Handlebars in via `getRenderer()` |
| [`form-preview/`](form-preview/) | Live form → preview with shared store and event handling |

---

> On release, the `examples/` folder is packaged as a ZIP download so you can grab it and
> start experimenting without cloning the full repository.
