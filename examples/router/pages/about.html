About

This example demonstrates SWC's built-in client-side router. Route content is loaded from plain HTML files using the src attribute — no component definition needed per page.

Four custom elements make it work: <router-container> owns the store, <router-link> handles navigation, <router-switch> renders the matching route, and <router-route> maps a path to a file.

The router is an optional add-on — the core library works perfectly without it.
